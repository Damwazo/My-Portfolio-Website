Forkify App

Projet N°11

Et voici une application web de recettes culinaires : Recherche par mots-clés (ingrédients, nom d’une recette), affichage dynamique des recettes dans l’interface utilisateur.

Vos recherches seront affichées dans le panneau gauche de l’application ; au-delà de dix résultats de recherches, ces derniers seront répartis sur autant de pages que nécessaire. Une recette vous plaît et vous désirez la garder en mémoire ? Ajoutez la en favori, et retrouvez-la plus tard dans votre liste de favoris (« bookmarks »). Enfin, vous pouvez ajouter vos propres recettes ; cela ouvrira un formulaire à remplir dans une fenêtre modale.

Si vous voulez tester l'application, suivez ce lien;
Le code de cette application est disponible sur ce repo GitHub.

Notes :
- Les recherches possibles sont limitées à cette liste;
- L’API connecté à ce projet ne peut supporter que 100 requêtes par heure.
