Bankist App

Projet N°9

Voici une interface en ligne pour l'application bancaire du service "Bankist". Elle recense tous les mouvements d'argent du client, en affichant le solde total du compte. Trois fonctionnalités sont disponibles et actives : Le Transfert d'argent vers un autre compte client de cette banque, la demande de prêt et la fermeture du compte affiché. Les mouvements d'argent sont affichés et mis à jour en temps réel et un calcul d'intérêt est également mis en place.

Pour utiliser l'application, deux comptes utilisateurs factices ont été créés : Les identifiants du premier compte sont les suivants :
user = js et pin = 1111.
Les identifiants du deuxième compte sont :
user = jd et le code pin = 2222.
Un minuteur est également intégré; si vous ne touchez plus à rien sur la page, vous serez automatiquement déconnecté.

Vous trouverez un organigramme et le code de cette application web vers ce repo GitHub, et la version live à essayer ici.
